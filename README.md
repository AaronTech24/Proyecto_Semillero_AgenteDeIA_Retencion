# Proyecto_Semillero_AgenteDeIA_Retencion
Este es un proyecto del semillero de netlife en la que se desarrolla un Agente de IA que se encargue de retener al cliente/usuario para que no dé de baja el servicio y a su vez genere soluciones, ya sea tecnico o económico y presentar un informe por si la situación empeora.
